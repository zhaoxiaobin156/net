How to capture HTTP conversations

To begin a capture:
  • select the appropriate* network adapter from the drop-down at the top right of the screen
  • press the Scoop button
  • to ensure that the latest requests are visible, even if they scroll off the bottom of the list, select the Follow button

As HTTP requests are made, new items appear in the list telling you what URLs are being accessed, the size and type of the returned file (where applicable), the HTTP response code and so on.

To get more details about a request, double-click an item from the list and view the information available in the various tabs in the pop-up window.

* Note that if you want to see conversations where the machine running HTTP Scoop is both the client and the server, you must select the Local Interface (lo0) from the network adapter drop-down.
