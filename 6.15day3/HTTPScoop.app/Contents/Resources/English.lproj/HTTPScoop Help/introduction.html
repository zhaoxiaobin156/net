An introduction to HTTPScoop

HTTP Scoop is an HTTP protocol analyser a.k.a. "HTTP sniffer". It passively captures HTTP conversations and presents their details in real-time.

HTTP Scoop is a great aid in diagnosing problems with web sites and web applications whatever client or server technology you use. Any type of HTTP traffic may be captured: HTML, XML, images, executables etc. You can see exactly what data is being exchanged and useful information is clearly presented to you such as request/response headers, request parameters and POST data. You can view responses as text or hexadecimal and save them to disk with a couple of clicks.

Happy scooping!

  • RFC 2616 - Hypertext Transfer Protocol -- HTTP/1.1, is the definitive reference for the HTTP protocol. Highly recommended reading.
  • James Marshall's excellent HTTP Made Really Easy gives a practical, easy-to-understand treatment of the protocol.
