Comment capturer des conversations HTTP

Pour commencer une capture :
  • Sélectionnez l'adaptateur réseau approprié* dans le menu déroulant en haut à droite.
  • Cliquez sur le bouton Scoop.
  • Cliquez sur le bouton Follow pour vous assurez que les dernières requêtes sont visibles.

Au fur et à mesure que les requêtes HTTP sont effectuées, des nouveaux éléments apparaissent dans la liste, vous informant de quelles URL sont chargées, de la taille et du type de fichier retourné, etc...

Pour obtenir plus de détails à propos d'une requête, double-cliquez sur un élément de la liste et affichez les informations disponibles dans les différents onglets de la fenêtre.

*Notez que si vous voulez afficher les conversations dans le cas où la machine lançant HTTPScoop est à la fois le client et le serveur, vous devez sélectionner l'adaptateur réseau Local Interface (lo0).
