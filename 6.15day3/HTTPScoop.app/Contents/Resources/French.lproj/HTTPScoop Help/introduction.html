Introduction à HTTPScoop

HTTPScoop est un analyseur de protocole HTTP, aussi appelé "Sniffeur HTTP". Il capture passivement des les conversations HTTP et les affiche en temps réel.

HTTPScoop aide à diagnostiquer les problèmes avec les sites et applications web quelque soit le client ou la technologie serveur utilisés. Il est possible de capturer n'importe quel type de trafic HTTP : HTML, XML, images, exécutables, etc... Vous pouvez voir exactement quelles données sont échangées et des informations utiles vous sont présentées : en-têtes, paramètres de la requête, données POST, etc... Vous pouvez afficher les réponses au format texte ou hexadécimal et les enregistrer sur votre disque.

Happy scooping !
